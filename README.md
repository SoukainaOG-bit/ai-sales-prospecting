# 🤖 AI Sales Prospecting Automation

Projet d'automatisation de prospection commerciale avec IA, scraping LinkedIn, enrichissement de données, qualification IA, et intégration CRM (HubSpot), via n8n.

## 🧩 Fonctionnalités

- Scrapping LinkedIn (via Phantombuster ou Puppeteer)
- Enrichissement des leads (Apollo.io / Hunter.io)
- Qualification IA : Lead Score + OpenToOutsourceScore
- Alimentation automatique dans HubSpot
- API Flask pour modèles de machine learning
